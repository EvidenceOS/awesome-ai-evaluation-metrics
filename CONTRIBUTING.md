# Contributing to Awesome AI Evaluation Metrics

Thank you for contributing! This list curates AI evaluation metrics across all modalities.

## How to Contribute

1. **Fork** this repository
2. **Add** your resource in the appropriate section
3. **Follow** the format: `- [Resource Name](URL) — Brief description`
4. **For structured data:** Add JSON entries to the `data/` directory following the existing schema
5. **Submit** a pull request with a clear description

## Quality Criteria

- Metrics should reference **original papers** or **widely adopted implementations**
- Include the **mathematical definition** or link to one where possible
- Benchmark resources should note **known limitations** (saturation, interface mismatch, etc.)
- Existing repo surveys should include **scope** and **gap analysis**

## Contribute & Publish

Top contributors are eligible for co-authorship on companion papers. See the README for details on our Human vs AI curation study.

## Code of Conduct

Be respectful, constructive, and focused on improving the resource for the community.
